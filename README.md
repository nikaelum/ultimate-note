# SmartDoc AI
AI-powered document editor using Google Gemini.
Paste raw text → AI formats it → renders as a 
clean printable document.

## Setup
1. Clone repo
2. Add your Gemini API key in the app settings
3. Deploy to Vercel (static site, no backend needed)

## Tech Stack
- Vanilla HTML/CSS/JS
- Google Gemini API (gemini-2.0-flash-lite)
- KaTeX for math rendering
- Marked.js for markdown
